# 🚨 AI Incident Investigation Agent

An end-to-end AI-powered incident monitoring and investigation platform designed to detect anomalies, analyze incidents, identify root causes, and generate actionable reports.

## 📌 Overview

AI Incident Investigation Agent helps organizations proactively monitor systems and investigate operational issues using Machine Learning, FastAPI, and AI-powered analysis.

The platform combines anomaly detection, incident management, and AI-driven root cause analysis into a single solution.

## ✨ Features

### 🔍 Incident Detection
- Sensor anomaly detection
- Real-time incident identification
- Confidence-based severity classification

### 📊 Incident Monitoring

### 🤖 AI Investigation
- Root cause analysis

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/AishwariyaAI/AI-Incident-Investigation-Agent.git
cd AI-Incident-Investigation-Agent
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Backend

```bash
uvicorn app.main:app --reload
```

### Open Swagger

```text
http://localhost:8000/docs
```
